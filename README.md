# ASCII-ANIMATION

**ASCII-ANIMATION** adalah proyek seni ASCII bergerak yang berjalan di terminal menggunakan Python dengan library NumPy, Pillow dan Requests.

## Cara Install

```bash
git clone https://github.com/irfanihbro/ascii-animation.git
cd ascii-animation
pip install -r requirements.txt
cd mode-jam-terbang  # atau chill-guy
py app.py # atau python app.py
```
